## Declaration and Assignment
1. `var` declares a variable, optionally initializing it to a value
2. Declaration:
```javascript
var a
//you declare the variable(a)
```
3. Assignment:
```javascript
var a = 1
//you assign the value(1) to the variable(a)
```
